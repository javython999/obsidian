>git commit

```shell
git commit
```
* staging 파일들을 commit 한다.

>git commit -m "commit message"
```
git commit -m "init"
```
* staging 파일들을 메세지와 함께 commit 한다.

>git commit -am "commit message"
```shell
git commit -am "init"
```
* 모든 staging 파일들을 메세지와 함께 commit 한다.

>git log
```shell
git log
```
* commit log를 보여준다.
